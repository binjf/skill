# Windows 快捷键

Alt+Tab ；  窗口切换

Ctrl+Print Screen:截静态图

Alt+Print Screen: 只截当前活动的图，静态不会截到

Win+Print Screen: 截图并自动保存到我的图片下

win + shift + 左右箭头 把当前文件移动到左右屏 

Win + Pause 打开系统配置环境变量

F1帮助

F2改名

F3搜索

F4地址

F5刷新

F6切换
